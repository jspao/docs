# Typecho 博客系统

Typecho 是一款轻量级的开源博客程序，基于 PHP 开发，以简洁、高效著称。

[官方网站](https://typecho.org/)

## 特点

- 🪶 **轻量快速** - 仅 7 张数据表，运行效率高
- 🎨 **主题丰富** - 大量免费主题可供选择
- 🔌 **插件扩展** - 丰富的插件生态
- 📝 **Markdown** - 原生支持 Markdown 写作
- 🗄️ **数据库** - 支持 MySQL、SQLite、PostgreSQL

## 适用场景

- 个人博客
- 技术笔记
- 轻量级内容站点

## 与 WordPress 对比

| 特性 | Typecho | WordPress |
|------|---------|-----------|
| 体积 | 轻量 (~500KB) | 较重 (~50MB+) |
| 性能 | 快 | 中等 |
| 易用性 | 简单 | 丰富但复杂 |
| 扩展性 | 插件较少 | 插件极丰富 |
| 适用 | 个人博客 | 企业网站 |

## 快速开始

```bash
# 1. 下载程序
wget https://github.com/typecho/typecho/releases/latest/download/typecho.zip

# 2. 解压到网站目录
unzip typecho.zip -d /var/www/typecho

# 3. 配置权限
chmod -R 777 /var/www/typecho/usr

# 4. 访问安装向导
# http://your-domain.com/install.php
```
